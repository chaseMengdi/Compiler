使用须知：
1、本目录下的compiler.cpp为源程序，compiler.exe为可执行文件，直接运行可执行文件，然后输入需要编译的c文件路径即可。注意，需要输入绝对路径，而不是相对路径，路径和文件名最好不要含有中文。
2、本目录下的1.c 2.c 3.c是3个测试用例，分别测试编译器的各项功能。
3、本目录下的1.asm 2.asm 3.asm是3个测试用例编译后生成的asm文件，是可以在masm中转换成exe文件的。如果有需要，可以删掉asm文件重新尝试生成。本目录下生成的asm文件都可以使用masm 2015正常编译执行，不会报错。
